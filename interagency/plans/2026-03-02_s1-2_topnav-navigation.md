# ПЛАН — S1.2 Shell: TopNav (горизонтальная навигация)
Дата: 2026-03-02  
Промт: `interagency/prompts/2026-03-02_s1-2_topnav-navigation.md`
Статус: active  

## 1. Анализ задачи
Необходимо заменить вертикальный `Sidebar` на горизонтальный `TopNav`. 
- **Источник данных**: `apps/web/lib/consulting/navigation-policy.ts` (`CONSULTING_NAVIGATION`).
- **Стили**: Geist Font, белый фон, тонкие границы (`border-black/10`).
- **Группировка**: Объединение доменов в 5 логических групп (Урожай, CRM, Финансы, Коммерция, Настройки).

## 2. Декомпозиция работ

### Шаг 1: Создание TopNav.tsx
- Реализовать контейнер `TopNav` с логотипом.
- Написать маппинг `CONSULTING_NAVIGATION` -> Группы.
- Реализовать Dropdown-компонент для подменю (на базе `framer-motion` для плавности или чистый CSS/React).
- Использовать существующий `getVisibleNavigation` для фильтрации по ролям.

### Шаг 2: Модификация AppShell.tsx
- Удалить импорт и использование `Sidebar`.
- Изменить структуру Grid/Flex: `TopNav` сверху на всю ширину, под ним основной контент.
- Обеспечить, чтобы `LeftRaiChatDock` корректно прижимался к левому краю под `TopNav`.

### Шаг 3: Корректировка стилей
- Убрать фиксированную ширину `aside` (350px) из глобальных стилей или `AppShell`.
- Проверить адаптивность (минимально, чтобы не поехало на средних экранах).

## 3. Технические детали
- **Шрифт**: `font-geist`.
- **Веса**: `font-normal` (400) и `font-medium` (500). **bold запрещён**.
- **Цвета**: `text-[#171717]`, `bg-white`, `border-black/10`.
- **Иконки**: `lucide-react`.

## 4. План тестирования
1. `cd apps/web && pnpm exec tsc -p tsconfig.json --noEmit` — проверка типов.
2. Manual smoke:
   - Переходы между пунктами меню.
   - Корректность отображения под разными ролями.
   - Сохранение состояния чата при кликах в TopNav.

## 5. Definition of Done (DoD)
- Навигация работает.
- Sidebar удалён.
- UI соответствует `UI_DESIGN_CANON.md`.
- Типы вадидны.
- Чат не размонтируется.

# Межведомственная папка (Cursor ↔ Antigravity)
Назначение: единое место в репе, где мы фиксируем **промты**, **планы**, **отчёты** и **решения**, чтобы разработка была воспроизводимой и без “потерянных договорённостей”.

## Структура
- `interagency/INDEX.md` — оглавление и быстрые ссылки на актуальные материалы
- `interagency/prompts/` — промты, которые ты кидаешь в Antigravity (кодинг)
- `interagency/plans/` — планы/декомпозиции (что и зачем делаем)
- `interagency/reports/` — отчёты по итогу прогонов (что сделано, что сломалось, что дальше)
- `interagency/templates/` — шаблоны для единообразия

## Нейминг файлов (чтобы не было помойки)
Формат: `YYYY-MM-DD_<slug>.md`
- `slug` — коротко, по делу: `p0-1_api-rai-chat`, `agro_draft_commit`, `workspace_context` и т.п.

## Правила (жёстко)
1) Один документ = одна тема/задача.  
2) В промте всегда есть **DoD**, **запреты**, **тест-план**, **что вернуть на ревью**.  
3) В отчёте всегда есть: что изменено, что запущено, что упало, ссылки на файлы/диффы.  
4) Никаких “вроде работает”. Только проверяемые факты.

# Готовые тексты для внутреннего оператора: регистрация и setup хозяйства

## Как использовать

- Этот файл нужен для ручного ввода во внутренний чат или для ручного переноса в UI.
- Все поля `TODO` заполняются пользователем по реальным реквизитам.
- Гл. агроном хозяйства здесь — внешний пользователь `front-office`, а не `chief_agronomist`.

## 8. Внутренний комментарий оператору

Если чат не сможет выполнить весь контур одним сообщением, раскладывай путь на отдельные шаги:

1. `party`
2. `account`
3. `contact`
4. `front-office invitation`
5. `field`
6. `season`
7. `harvestPlan`

Главная цель setup-части — не “красивый диалог”, а появление полного минимального контура для генерации `TechMap`.

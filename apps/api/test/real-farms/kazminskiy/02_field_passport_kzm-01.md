# Паспорт тестового поля `KZM-01`

## Роль поля в тесте

`KZM-01` — это один крупный производственный массив для первого полного ручного прогона `TechMap` под озимый рапс.

Поле регистрируется в системе как **одна сущность**, даже несмотря на внутреннюю неоднородность. Это сделано специально, чтобы проверить:

- работу техкарты на большом поле порядка `1000 га`;
- влияние масштаба массива на сроки операций;
- дозапрос недостающего контекста;
- explainability по неоднородному, но не экстремальному полю.

## Общая характеристика

| Поле | Значение |
|---|---|
| Код поля | `KZM-01` |
| Рабочее имя | `Казьминский-Рапс-01` |
| Площадь | `1006 га` |
| Тип массива | крупный производственный массив |
| Культура | `озимый рапс` |
| Сезон | `2026/2027` |
| Географическая привязка | территория рядом с `с. Казьминское`, Кочубеевский район, Ставропольский край |
| Формат регистрации в системе | одно поле |
| Фактическая неоднородность | 3 внутренние агрозоны |

## Рельеф и агрофон

| Параметр | Значение |
|---|---|
| Общий рельеф | слабоволнистый |
| Уклон | от `0.5%` до `2.8%` |
| Экспозиция | преимущественно юго-восточная и южная на более сухой части массива |
| Риск водной эрозии | низкий до умеренного |
| Дренированность | в целом хорошая, локально уплотнение и переувлажнение после ливней в понижениях |
| Каменистость | практически отсутствует |
| Проходимость техники | хорошая большую часть сезона, но после интенсивных осадков есть локальные ограничения в понижениях |

## Предшественник и история культуры

| Параметр | Значение |
|---|---|
| Предшественник | `озимая пшеница` |
| Уборка предшественника | конец июня – начало июля |
| Основной фон после уборки | достаточный запас пожнивных остатков, локально неравномерное распределение соломы |
| Рапс в структуре севооборота | не чаще 1 раза в `4 года` |
| Ближайший рапсовый предшествующий цикл на этом массиве | `2022` |
| Критичный фитосанитарный фон по крестоцветным | не зафиксирован как жёсткий blocker, но контроль падалицы и сорняков обязателен |

## Внутренние агрозоны массива

Поле регистрируется как одно, но при интерпретации контекста и explainability нужно держать в голове три внутренние зоны.

### Агрозона A — северо-западная

| Показатель | Значение |
|---|---|
| Площадь | `318 га` |
| Особенность | более ровный микрорельеф и лучшая влагоудерживающая способность |
| Почвенный фон | тяжёлый суглинок, чернозёмный профиль |
| Риск | локальное переуплотнение после проходов тяжёлой техники |

### Агрозона B — центральная

| Показатель | Значение |
|---|---|
| Площадь | `411 га` |
| Особенность | основной производственный контур |
| Почвенный фон | средний суглинок, наиболее типичный для массива |
| Риск | неравномерность по сере и бору |

### Агрозона C — южно-восточная

| Показатель | Значение |
|---|---|
| Площадь | `277 га` |
| Особенность | более открытая, быстрее пересыхает в августе |
| Почвенный фон | более лёгкий суглинок |
| Риск | более слабые всходы при дефиците стартовой влаги |

## Ограничения по срокам и проходу техники

| Ограничение | Как влияет на техкарту |
|---|---|
| Масштаб поля `~1000 га` | требует высокой дисциплины по посевному окну |
| Один массив, но 3 агрозоны | часть поля может войти в зиму неравномерно |
| Ограниченное число посевных агрегатов | риск растяжки посева на `6–8 дней` вместо желательных `4–5` |
| Один основной опрыскиватель | риск смещения сроков фунгицидно-регуляторных обработок |
| Неравномерное состояние после предшественника | может ухудшать однородность появления всходов |

## Вывод для `TechMap`

Для этой техкарты поле нельзя трактовать как маленькое однородное демо-поле. Это крупный реальный производственный массив, где:

- агроусловия в целом подходят под озимый рапс;
- почва в базе рабочая, но не идеальная;
- организационная дисциплина и техника являются частью агрономического риска;
- `ControlPoint` по осенней розетке и перезимовке обязателен не формально, а по сути сценария.

# Тестовый пакет хозяйства «СПК колхоз племзавод „Казьминский“»

## Назначение

Этот пакет нужен для ручного `end-to-end` теста полного пути:

1. регистрация нового реального хозяйства, которого нет в базе;
2. создание связанных сущностей для `TechMap`;
3. приглашение внешнего гл. агронома хозяйства во `front-office`;
4. первичный контекст от хозяйства;
5. дозапрос недостающих данных через `clarification loop`;
6. генерация техкарты по озимому рапсу.

Целевой сценарий:

- хозяйство: `СПК колхоз племзавод «Казьминский»`
- культура: `озимый рапс`
- сезон: `2026/2027`
- тестовое поле: `KZM-01`
- масштаб поля: около `1000 га`

## Что в пакете реально, а что синтетика

### Реальные данные

- наименование хозяйства;
- адрес хозяйства;
- территориальная привязка: `с. Казьминское, Кочубеевский район, Ставропольский край`;
- координатный и климатический baseline территории;
- зональные почвенно-агроклиматические ориентиры по Ставропольскому краю и району.

### Синтетические, но правдоподобные данные

- паспорт тестового поля `KZM-01`;
- структура внутренних агрозон внутри массива;
- техника и организационные ограничения;
- результаты анализа почвы;
- операционный и ресурсный контекст хозяйства;
- готовые тексты для внутренних и внешних чатов;
- ручной тестовый сценарий.

### Что вводит пользователь вручную

- полные юридические реквизиты;
- `ИНН`, `КПП`, `ОГРН`, банковские реквизиты;
- `ЛПР` со стороны хозяйства;
- реальный гл. агроном хозяйства;
- `Telegram ID`, телефон, email и иные контактные данные.

## Базовые правила использования

- `chief_agronomist` не является гл. агрономом хозяйства.
- Гл. агроном хозяйства в этом сценарии — внешний пользователь `front-office`.
- Основной внешний канал теста: `front-office web chat`.
- `Telegram` допустим как дополнительный канал, но не как единственный обязательный канал ручного теста.
- Контекст должен собираться не анкетой целиком, а по модели `gap-driven clarification`.

## Ключевые источники территориальной основы

1. `Mapcarta / OpenStreetMap` — координаты населённого пункта `Казьминское`: `44.59216, 41.67778`.
2. `Wikipedia` — территориальная привязка `Кочубеевского района` и общая география района.
3. `Гидрометцентр России / Meteoinfo` — климатические нормы по `Невинномысску` как ближайшему реперному климатическому узлу для данной территории.
4. `Climate-Data.org` — современная климатическая сводка по `Ставрополю` как региональному бенчмарку по распределению осадков и температур.
5. Открытые региональные материалы по географии и почвам Ставропольского края — для выбора правдоподобного чернозёмного профиля.

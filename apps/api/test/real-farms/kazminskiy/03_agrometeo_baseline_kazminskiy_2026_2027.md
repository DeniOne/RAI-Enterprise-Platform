# Территориальный агро-метео baseline для Казьминского под озимый рапс 2026/2027

## Что это за baseline

Это не локальная метеостанция прямо внутри поля, а территориальная база для правдоподобного теста хозяйства в районе `с. Казьминское`.

Документ разделяет:

- **внешне подтверждённые факты**;
- **агрономические выводы для тестового сценария озимого рапса**.

По открытым материалам по географии и почвам Ставропольского края:

- в крае широко распространены чернозёмы, прежде всего `обыкновенные` и `южные`;
- для более возвышенных и центрально-западных частей региона встречаются чернозёмные профили с более благоприятным потенциалом для полевых культур;
- для тестового сценария около Казьминского правдоподобно использовать чернозёмный суглинистый контур, а не каштановый засушливый профиль.

## Агро-метео интерпретация для озимого рапса

Ниже — уже не “голые данные”, а рабочая агрономическая интерпретация для тестового сценария.

### 1. Посевное окно

Для территории типа `Казьминское / запад Ставропольского края` рабочим окном под озимый рапс в тесте принимается:

- базовое окно: `20 августа — 5 сентября`
- допустимое растянутое окно при больших площадях: до `8–10 сентября`

Почему:

- август и начало сентября остаются тёплыми;
- до устойчивых холодов обычно хватает времени на развитие осенней розетки;
- главным ограничителем становится не холод, а стартовая влага и организационная скорость посева.

### 2. Главный осенний риск

Ключевой риск для тестового сценария — **не холод, а неравномерная влагообеспеченность при посеве на крупном массиве**.

Практический эффект:

- часть поля может дать быстрые дружные всходы;
- более сухая часть поля войдёт в осень слабее;
- при растянутом окне посева массив может войти в зиму неодинаково развитым.

### 3. Осеннее развитие розетки

Для этой территории озимый рапс реалистично можно вывести в рабочую розетку до зимы, если одновременно выполняются три условия:

1. посев укладывается в основное окно;
2. стартовое питание даётся без провала по фосфору и сере;
3. на поле не возникает сильной неравномерности по влаге.

Отсюда обязательность `ControlPoint`:

- `Контроль осенней розетки`

### 4. Риск перезимовки

Территория не выглядит экстремально жёсткой по зиме для озимого рапса, но риск перезимовки в тесте должен оставаться значимым из-за:

- неоднородности массива;
- вероятной разницы в развитии растений на момент ухода в зиму;
- возможной слабости части поля по бору, сере и корневому развитию.

Отсюда обязательность `ControlPoint`:

- `Контроль перезимовки`

### 5. Весенний старт

Весеннее возобновление вегетации для такой территории обычно даёт хороший потенциал, если растения ушли в зиму неослабленными. Но для большого поля это означает:

- азотно-серное питание должно быть очень дисциплинированным по срокам;
- один слабый организационный контур по технике легко превращается в потери урожайности;
- весеннее окно по осмотру и первому питанию на `1000 га` уже критично зависит от логистики.

### 6. Как масштаб поля влияет на техкарту

На поле `~1000 га` климатический риск и организационный риск почти сливаются.

Для этой техкарты это означает:

- посевное окно должно считаться не абстрактно, а через пропускную способность техники;
- осенняя оценка розетки нужна маршрутная, а не по одной точке;
- весной перезимовка оценивается по зонам массива;
- на explainability стороне система должна видеть, что риск растяжки операций связан не только с погодой, но и с площадью массива.

## Практический вывод для теста

Территория `Казьминского` для озимого рапса выглядит правдоподобной и рабочей. Тестовая логика не должна моделировать катастрофический климатический сценарий. Она должна моделировать **рабочий производственный сценарий с нормальным потенциалом культуры, но с реальными рисками по влаге, срокам и неоднородности большого массива**.

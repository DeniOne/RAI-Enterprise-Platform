# Project Brief: RAI_EP

## Core Mission
RAI_EP — это операционная среда для управления агробизнесом (RAI), построенная на базе универсального бизнес-ядра (Business Core). Цель проекта — извлечь чистую бизнес-логику из MatrixGin и адаптировать её под нужды современного агропрома, сохраняя принципы ясности, порядка и человекоцентричности.

## Fundamental Principles (Constitutional L0)
- **Core-First**: Ядро (Business Core) первично и универсально. Доменная логика (RAI) подключается отдельно.
- **Canon > Code**: Документация (Канон) является высшим авторитетом. Код должен следовать Канону буквально.
- **Waste Elimination**: Устранение потерь в процессах через автоматизацию и прозрачность реактивных механизмов.
- **Sovereignty**: Система должна обеспечивать суверенитет данных и процессов владельца бизнеса.

## Objectives
- Экстракция Business Core из MatrixGin.
- Создание доменного слоя RAI для управления агропроизводством.
- Реализация адаптерного слоя для интеграции с внешними интерфейсами (Telegram и др.).
- Построение системы на событийной модели и FSM.

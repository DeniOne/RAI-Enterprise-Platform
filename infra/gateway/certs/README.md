# Gateway Certificates

Этот каталог зарезервирован для локально смонтированных сертификатов gateway и не должен содержать реальные закрытые ключи в Git.

Правило:
- реальные `*.key`, `*.crt`, `*.pem` подставляются через secret storage или локальный mount вне репозитория
- в Git разрешены только `README.md`, `.gitignore` и при необходимости `*.example`

Минимально ожидаемые runtime-артефакты для локального/ops окружения:
- `server.crt`
- `server.key`
- `ca.crt`

Если нужен локальный стенд с TLS, создайте временные сертификаты вне Git и смонтируйте их в контейнер при запуске.

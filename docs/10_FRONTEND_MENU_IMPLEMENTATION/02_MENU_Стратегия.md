## 1. Бизнес-роль

Контур руководства: портфель, риск-карта, сценарии, журнал стратегических решений.

## 2. Целевое дерево страниц

- `/strategy`
- `/strategy/overview`
- `/strategy/portfolio`
- `/strategy/risks`
- `/strategy/scenarios`
- `/strategy/log`

## 3. Фактический статус

- Реализована базовая страница `/strategy`.
- Детальные разделы отсутствуют.

## 4. API-связки

- `/api/strategic*`
- `/api/risk*`
- `/api/consulting/strategic/dashboard/*`
- `/api/consulting/simulations/*`

## 5. Критерий готовности MVP

- Руководитель видит целостную панель: портфель планов, активные риски, 2-3 сценария и журнал решений.

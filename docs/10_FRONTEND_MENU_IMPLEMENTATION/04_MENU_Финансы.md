# Меню 4: Финансы

## 1. Бизнес-роль

Контур финансовой дисциплины: денежные потоки, начисления, расчёты, отчётность.

## 2. Целевое дерево страниц

- `/finance`
- `/finance/cashflow`
- `/finance/performance`
- `/finance/invoices`
- `/finance/reporting`

## 3. Фактический статус

- Есть только `/finance` (заглушка).
- Подроуты отсутствуют.

## 4. API-связки

- `/api/ofs/finance/dashboard`
- `/api/consulting/budgets/*`
- `/api/consulting/decisions/*` (для подтверждений)
- backend finance/economy события и агрегаты

## 5. Критерий готовности MVP

- Пользователь видит баланс, бюджетный burn rate, ключевые начисления и список финансовых событий.

# Меню 6: Производство

## 1. Бизнес-роль

Контур физического производства (Грипил): закупка, хранение, производство, качество, логистика.

## 2. Целевое дерево страниц

- `/production`
- `/production/procurement`
- `/production/storage`
- `/production/manufacturing`
- `/production/quality`
- `/production/logistics`
- `/production/analytics`

## 3. Фактический статус

- Есть только `/production` (заглушка).
- Подроуты отсутствуют.

## 4. API-связки

- На текущем этапе использовать данные из `registry`, `knowledge`, `economy` и профильных production-модулей по мере их появления.
- Если backend-эндпойнты не готовы, временно показывать read-model с явной маркировкой источника.

## 5. Критерий готовности MVP

- Минимум: список производственных партий/процессов, статус качества, этап логистики.

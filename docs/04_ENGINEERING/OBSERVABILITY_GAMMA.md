# Observability для Gamma

## Фокус
- Мониторинг производительности и доступности AI-сервисов.
- Логирование бизнес-решений.
- Трассировка сигналов от входа до совета.

## Цели и инструменты
- Сквозной Trace ID для каждого запроса.
- Полный Audit Trail в БД для отчетности.

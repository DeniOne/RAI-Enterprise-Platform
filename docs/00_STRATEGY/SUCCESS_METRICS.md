# Метрики успеха и KPI 📊

Как мы измеряем эффективность внедрения RAI Enterprise Platform.

---

## 1. Ценность для Клиента (Фермера)

### 1.1 Урожайность (Yield)
- **Цель**: +15% к среднему показателю за 3 года.
- **Метод измерения**: Сравнение чеков приемки с историческими данными.

### 1.2 Себестоимость (OpEx)
- **Цель**: -10% затрат на ГСМ и СЗР за счет точного планирования.
- **Метод измерения**: План-фактный анализ техкарт.

### 1.3 Скорость реакции (SLA)
- **Цель**: < 4 часов от обнаружения проблемы (через спутник/обход) до постановки задачи.
- **Метод измерения**: Время жизни задачи в статусе `CREATED`.

---

## 2. Эффективность системы (RAI EP)

### 2.1 Активность (DAU/MAU)
- **Цель**: 80% агрономов хозяйства должны использовать систему ежедневно в сезон.
- **Метод измерения**: Логи аутентификации Identity Service.

### 2.2 Доверие к ИИ (Advisory Accuracy)
- **Цель**: 70% рекомендаций системы должны приниматься агрономом без изменений.
- **Метод измерения**: Статистика подтверждения "намерений" (Intents).

---

## 3. Технические метрики

### 3.1 Надежность данных
- **Цель**: 0% потерь событий аудита даже при сбое БД.
- **Метод измерения**: Мониторинг таблицы `AuditFailure`.

### 3.2 Качество геометрии
- **Цель**: Точность границ полей < 1 метра при сравнении с КН.
- **Метод измерения**: Валидация через PostGIS функции пересечения.

---

## 4. Критерии закрытия фаз

### Phase Alpha Success
- Минимум 2 реальных хозяйства в системе.
- Закрыт полный цикл "Посев-Уборка" хотя бы для одного поля.
- Пользователи подтвердили удобство Telegram-бота.

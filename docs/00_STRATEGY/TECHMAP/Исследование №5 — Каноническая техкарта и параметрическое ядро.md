## 1. Executive Synthesis

Настоящий документ — финальный синтез четырёх исследовательских модулей в единую архитектуру канонической техкарты, пригодную для цифровой системы RAI_EP.

**Архитектурные принципы:**

1. **Двуветвевая модель.** Существуют две самостоятельные техкарты — озимый и яровой рапс. Выбор ветки — первое решение системы.
2. **Неизменное ядро + адаптивная оболочка.** Каждая техкарта содержит инвариантные блоки (что делать) и параметрические (как именно делать, зависит от условий).
3. **Контрольные точки как decision gates.** В ключевых фазах система запрашивает данные и принимает/рекомендует решение.
4. **Каузальная цепь.** Каждая операция имеет цель, входные условия, параметры качества и связь с последующими операциями.

## 2. Общая логика идеальной техкарты

### 2.1. Что универсально (общее для озимого и ярового)

| Блок | Универсальные принципы |
|------|----------------------|
| Севооборот | Минимум 4 года без рапса/крестоцветных |
| pH | Оптимум 6,0–7,0; <5,5 — абсолютное противопоказание |
| Посевное ложе | Мелкокомковатость, выровненность, уплотнённое дно |
| Прикатывание | Обязательно после посева |
| Питание S | 20–40 кг S/га — обязательно |
| Питание B | 200–400 г B/га листовые — обязательно |
| Протравливание | Инсекто-фунгицидное — обязательно |
| Пороговая защита | Мониторинг + ЭПВ вместо шаблона |
| Рапсовый стол | Обязательное оснащение при уборке |
| Уборка утром/вечером | Снижение потерь от хрупкости стручков |

### 2.2. Что ветвится по форме культуры

| Параметр | Озимый | Яровой |
|----------|--------|--------|
| Срок сева | Август — начало сентября | Апрель — май |
| Норма высева | 0,5–1,0 млн шт./га | 1,5–2,5 млн шт./га |
| Осенний блок | Развитие розетки, регуляторы, подготовка к зиме | Отсутствует |
| Зимовка | Критический этап | Отсутствует |
| Весеннее возобновление | Оценка перезимовки, решение | Посев + старт |
| Количество N-подкормок | 2–3 (осень + 2 весенних) | 1–2 (основная + подкормка) |
| Общая доза N | 120–200 кг/га | 90–140 кг/га |
| Регуляторы роста | Осень (обязательно) + весна | Обычно не требуются |
| Критическая фаза | Перезимовка | Цветение (засуха/жара) |

### 2.3. Что ветвится по типам условий

| Условие | Что адаптируется |
|---------|-----------------|
| Агроклиматическая зона (САТ, ГТК) | Форма культуры, сорт, сроки |
| Тип почвы (гранулометрия) | Система обработки, глубина |
| Уплотнение | Глубокое рыхление — да/нет |
| Влагозапас | Влагосбережение, сроки сева |
| Тип зимы | Зимостойкость сорта, регуляторы |
| Фитосанитарная история | Допуск/запрет сева, защита |
| Обеспеченность NPK/S/B | Дозы удобрений |
| Засорённость | Схема гербицидной защиты |

## 3. Каноническая техкарта озимого рапса

### Этап 1: Подготовка поля

| Атрибут | Значение |
|---------|---------|
| **Цель** | Создать идеальные стартовые условия |
| **Агрологика** | Фундамент всей технологии; ошибки здесь некомпенсируемы |
| **Входные условия** | Результат агрохиманализа (pH, NPK, S, B), история поля, предшественник |
| **Обязательные операции** | Лущение стерни → внесение P/K → основная обработка → предпосевная → выравнивание |
| **Окно** | Июль — начало августа (сразу после уборки предшественника) |
| **Критические параметры** | pH ≥6,0; отсутствие уплотнения; предшественник ≠ крестоцветные; ротация ≥4 года |
| **Контроль качества** | Мелкокомковатость ложа (1–5 мм), выровненность (±1 см), глубина обработки |
| **Типовые ошибки** | Сев на кислой почве; плужная подошва; грубое ложе; сев после рапса через 2–3 года |
| **Ветвится** | Система обработки (вспашка / mini-till / strip-till) — по типу почвы |
| **Неизменно** | Требование к pH, ротации, качеству ложа |

### Этап 2: Посев

| Атрибут | Значение |
|---------|---------|
| **Цель** | Равномерные своевременные всходы |
| **Агрологика** | Каждый день опоздания = потеря зимостойкости |
| **Входные условия** | t почвы, влага, готовность поля, дата устойчивого перехода t через +5°C (осень) |
| **Обязательные операции** | Протравливание → калибровка сеялки → посев → прикатывание |
| **Окно** | 90–110 дней до устойчивых заморозков (зонально: 10 авг — 10 сент) |
| **Критические параметры** | Глубина 1,5–2,5 см (±0,5); норма 0,5–1,0 млн шт./га; равномерность CV <15% |
| **Контроль** | Фактическая глубина (замер), густота всходов (подсчёт) |
| **Ошибки** | Поздний сев; глубокая заделка; нет прикатывания; неподходящий сорт |
| **Ветвится** | Срок (по зоне), норма (сорт vs гибрид), сорт (по зимостойкости) |
| **Неизменно** | Протравливание, прикатывание, диапазон глубины |

### Этап 3: Осенний уход

| Атрибут | Значение |
|---------|---------|
| **Цель** | Оптимальная розетка к зиме (6–8 листьев, шейка ≥8 мм, точка роста ≤2 см) |
| **Агрологика** | Состояние к зиме определяет >50% итогового урожая |
| **Операции** | Мониторинг всходов → инсектицид (блошки) → гербицид → N стартовый (20–40 кг) → регулятор роста + фунгицид (BBCH 14–16) → B подкормка → оценка готовности к зиме |
| **Окно** | Сентябрь — октябрь |
| **Критические параметры** | Густота ≥30 раст./м²; розетка 6–8 листьев; шейка ≥8 мм; точка роста ≤2 см |
| **Ошибки** | Избыток N (>50 кг → перерастание); нет регулятора; пропуск блошек; нет оценки к зиме |
| **Ветвится** | Доза N (по почвенному анализу); необходимость регулятора (по состоянию посева) |
| **Неизменно** | Обязательная оценка к зиме; борная подкормка; протравливание |

### Этап 4: Зимний покой

| Атрибут | Значение |
|---------|---------|
| **Цель** | Максимальное выживание |
| **Операции** | Мониторинг (снежный покров, t min, ледяная корка) |
| **Контроль** | Фактические температуры vs зимостойкость сорта; наличие снега |
| **Ветвится** | Нет операций; только мониторинг и прогнозирование |

### Этап 5: Весеннее возобновление

| Атрибут | Значение |
|---------|---------|
| **Цель** | Быстрая регенерация, решение сохранять/пересевать |
| **Операции** | Оценка перезимовки → решение → первая N подкормка (80–120 кг) + S (20–30 кг) |
| **Окно** | Немедленно при возобновлении вегетации (по мёрзло-талой, если возможно) |
| **Критические параметры** | % живых растений (≥70% от осенней густоты = сохранять; <40% = пересев) |
| **Ветвится** | Доза N (по густоте, по анализу); решение сохранять/пересевать |

### Этап 6: Стеблевание — бутонизация

| Атрибут | Значение |
|---------|---------|
| **Цель** | Максимальное ветвление, закладка стручков |
| **Операции** | Вторая N подкормка (40–60 кг) → регулятор роста (при необходимости) → инсектицид (скрытнохоботник, по порогу) → B подкормка (BBCH 50–51) |
| **Окно** | Апрель — начало мая |
| **Ветвится** | Доза N; необходимость регулятора; B доза |

### Этап 7: Цветение

| Атрибут | Значение |
|---------|---------|
| **Цель** | Максимальное завязывание |
| **Операции** | Инсектицид (цветоед, BBCH 51–59, ДО раскрытия!) → фунгицид (склеротиниоз, начало цветения) |
| **Критические параметры** | t <30°C; достаток влаги; B в норме; нет инсектицида в цветение (пчёлы!) |
| **Ветвится** | Фунгицид (по прогнозу влажности / модели болезней) |

### Этап 8: Налив — созревание — уборка

| Атрибут | Значение |
|---------|---------|
| **Цель** | Сохранение сформированного урожая |
| **Операции** | Мониторинг болезней стручков → десикация (побурение 70–80%) → прямое комбайнирование (влажность 9–12%) → сушка → хранение |
| **Окно** | Июнь — июль |
| **Критические параметры** | Рапсовый стол; уборка утром/вечером; скорость 4–6 км/ч |
| **Ошибки** | Перестой >7 дней; нет стола; уборка в жару |
| **Неизменно** | Рапсовый стол; утренняя/вечерняя уборка |

## 4. Каноническая техкарта ярового рапса

### Этап 1: Подготовка (осень предыдущего года)

| Атрибут | Значение |
|---------|---------|
| **Цель** | Влагонакопление + идеальные стартовые условия |
| **Операции** | Лущение стерни → агрохиманализ → P/K под зябь → зяблевая обработка → снегозадержание (Сибирь) |
| **Критические параметры** | pH ≥6,0; ротация ≥4 года; осенняя обработка (НЕ весенняя вспашка!) |

### Этап 2: Предпосевная подготовка (весна)

| Атрибут | Значение |
|---------|---------|
| **Цель** | Влагосбережение + посевное ложе |
| **Операции** | Закрытие влаги (боронование) → внесение N+S (основная доза) → предпосевная культивация на глубину сева |
| **Окно** | За 3–5 дней до сева |
| **Критическоe** | Минимальная весенняя обработка; запрет весенней вспашки |

### Этап 3: Посев

| Атрибут | Значение |
|---------|---------|
| **Цель** | Быстрые равномерные всходы |
| **Входные условия** | t почвы ≥+5°C (оптимум +8…+12°C); физическая спелость почвы |
| **Операции** | Протравливание → калибровка → посев (1,5–2,5 см) → стартовое P в рядок → прикатывание |
| **Окно** | Зонально: 2 дек. апреля (ЦЧР) — 3 дек. мая (Сибирь). Чем раньше — тем лучше! |
| **Критические параметры** | Норма 1,5–2,5 млн шт./га; глубина ±0,5 см; равномерность |
| **Ошибки** | Позднее; глубоко; без прикатывания; весенняя вспашка |

### Этап 4: Ранний рост (BBCH 10–19)

| Атрибут | Значение |
|---------|---------|
| **Цель** | Максимальный early vigour |
| **Операции** | Мониторинг всходов → инсектицид (блошки, немедленно по порогу!) → гербицид (BBCH 12–14) |
| **Критическоe** | Первые 3 недели — максимальная уязвимость; без протравки потери до 100% |

### Этап 5: Стеблевание — бутонизация (BBCH 30–59)

| Атрибут | Значение |
|---------|---------|
| **Операции** | N подкормка (30–50 кг) → инсектицид (по порогу) → B подкормка (BBCH 50–51) → инсектицид (цветоед, BBCH 51–59) |

### Этап 6: Цветение (BBCH 60–69)

| Атрибут | Значение |
|---------|---------|
| **Операции** | Фунгицид (склеротиниоз, начало цветения, по прогнозу) |
| **Критическоe** | Цветение совпадает с пиком жары (июль); яровой более уязвим чем озимый |

### Этап 7: Налив — уборка (BBCH 70–89)

| Атрибут | Значение |
|---------|---------|
| **Операции** | Десикация (70–80% побурения) → прямое комбайнирование (9–12%) → сушка → хранение |
| **Критическоe** | Рапсовый стол; утро/вечер; перестой = 15–25% потерь |

## 5. Сравнительная матрица: озимый vs яровой

| Параметр | Озимый | Яровой |
|----------|--------|--------|
| Число этапов техкарты | 8 | 7 |
| Общая длительность | 10–11 месяцев | 3–4 месяца |
| Число контрольных точек | 9 | 8 |
| Число N-подкормок | 3 (осень + 2 весны) | 2 (основная + 1 подкормка) |
| Общая доза N | 120–200 кг/га | 90–140 кг/га |
| Регуляторы роста | Обязательны (осень) | Обычно не нужны |
| Потенциал урожайности | 3,0–5,5 т/га | 1,8–3,5 т/га |
| Главный риск | Перезимовка | Засуха при цветении |
| Число адаптивных параметров | 12 | 10 |
| Требование к генетике | Зимостойкость | Скороспелость + early vigour |

## 6. Матрица ветвления по условиям

| Параметр | Стат/Дин | Обязат. | Влияет на блоки | Как меняет решение |

| **NPK обеспеченность** | Динам. | ✅ | Дозы удобрений | Расчёт по выносу: целевая урожайность × вынос − запас |

| **B в почве** | Динам. | ✅ | Листовая подкормка | Дефицит = 300–400 г; норма = 150–200 г |
| **Предшественник** | Динам. | ✅ | Допуск сева | Крестоцветные <4 года = запрет |
| **Уплотнение** | Динам. | ✅ | Обработка | Есть = глубокое рыхление; нет = стандарт |
| **Влагозапас (весна)** | Динам. | ✅ (яр.) | Решение о посеве | <100 мм = рапс рискован; >150 = оптимум |
| **Густота всходов** | Динам. | ✅ | Коррекция технологии | <15 раст./м² = пересев; 15–25 = усиленное питание; >25 = норма |
| **Состояние розетки (осень)** | Динам. | ✅ (озим.) | Регуляторы, прогноз зимовки | <5 листьев = слабый; 6–8 = норма; >10 = перерастание |
| **% перезимовки** | Динам. | ✅ (озим.) | Сохранять/пересевать | <40% = пересев; 40–70% = усиленный уход; >70% = норма |
| **Фитосанитарная история** | Динам. | ✅ | Защита, допуск | Кила в истории = запрет 7+ лет |
| **Метеопрогноз** | Динам. | ⚠️ | Сроки операций | Заморозки → задержка; дождь → ускорение/отсрочка |
| **Модель болезней** | Динам. | ⚠️ | Фунгицид цветение | Высокий риск = обработка; низкий = пропуск |

### A.2. Триггеры ветвления

```yaml
branching_triggers:
  - trigger: crop_form_selection
    input: [SAT_avg, winter_type, agroclimatic_zone]
    logic: |
      if SAT < 1700: REJECT ("рапс невозможен")
      if SAT 1700-2200: crop_form = SPRING
      if SAT > 2200 AND winter_type IN [mild, moderate]: crop_form = WINTER or SPRING
      if SAT > 2200 AND winter_type IN [severe, unstable]: crop_form = SPRING

  - trigger: field_admission
    input: [pH, rotation_years_since_rapeseed, predecessor, clubroot_history]
    logic: |
      if pH < 5.5: REJECT
      if rotation_years < 4: REJECT
      if predecessor IN [rapeseed, mustard, turnip_rape]: REJECT
      if clubroot_history AND rotation_years < 7: REJECT

  - trigger: tillage_system
    input: [soil_texture, compaction, drainage]
    logic: |
      if compaction = TRUE: deep_ripping = MANDATORY
      if soil_texture = clay: tillage = CONVENTIONAL
      if soil_texture = loam: tillage = MINIMUM or CONVENTIONAL
      if soil_texture = sandy_loam: tillage = MINIMUM or NO_TILL

  - trigger: variety_selection
    input: [crop_form, SAT_avg, winter_type, agroclimatic_zone]
    logic: |
      WINTER: prioritize winter_hardiness matching winter_type
      SPRING: prioritize maturity_group fitting SAT_available

  - trigger: sowing_window
    input: [crop_form, agroclimatic_zone, soil_temp, frost_forecast]
    logic: |
      WINTER: 90-110 days before first_frost_date_avg
      SPRING: soil_temp >= 5°C AND physical_ripeness = TRUE

  - trigger: N_dose_calculation
    input: [target_yield, N_soil, predecessor_N_credit, crop_form]
    logic: |
      total_N = target_yield * 55 - N_soil - predecessor_credit
      WINTER: split = [autumn: min(40, 0.2*total), spring1: 0.5*total, spring2: remainder]
      SPRING: split = [presowing: 0.65*total, topdress: 0.35*total]

  - trigger: overwintering_decision
    input: [survival_pct, plant_density_spring]
    logic: |
      if survival_pct >= 70% AND density >= 25/m²: CONTINUE
      if survival_pct 40-70% AND density 15-25/m²: CONTINUE + enhanced_nutrition
      if survival_pct < 40% OR density < 15/m²: RESOW_SPRING_CROP

  - trigger: sclerotinia_spray
    input: [sclerotinia_risk_model, precipitation_forecast, crop_stage]
    logic: |
      if bbch IN [61-65] AND risk_score > threshold: SPRAY
      else: SKIP

  - trigger: desiccation
    input: [pod_browning_pct, maturity_uniformity]
    logic: |
      if browning >= 70% AND uniformity = LOW: DESICCATE
      if browning >= 90% AND uniformity = HIGH: DIRECT_HARVEST
```

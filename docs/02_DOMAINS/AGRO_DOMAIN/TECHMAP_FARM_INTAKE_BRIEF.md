# Бриф хозяйства для разработки техкарты

## CLAIM
id: CLAIM-DOM-AGRO-TECHMAP-FARM-INTAKE-BRIEF-20260402

## Назначение

Это рабочий табличный бриф для сбора исходных данных от хозяйства перед разработкой техкарты.

## Как заполнять

| Поле | Правило |
|---|---|
| `Значение` | Указывается факт, число, дата, диапазон или один вариант из справочника |
| `Ед.` | Обязательна для всех числовых факторов |
| `Обязательность` | `P0` блокирует техкарту, `P1` снижает качество, `P2` полезно для уточнения |
| `Источник` | Протокол, карта, ERP, журнал, лаборатория, эксперт хозяйства |
| `Дата источника` | Обязательна для анализа, погодных и лабораторных данных |

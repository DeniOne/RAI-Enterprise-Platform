| Код | Фактор | Ед. | Почему критичен |
|---|---|---|---|
| RAI-001 | `pH` | pH | блок admission и риск килы |

| RAI-005 | сроки сева | дата | один из главных драйверов урожая |
| RAI-006 | развитие розетки к зиме | листьев, мм, см | ключ к перезимовке |
| RAI-007 | выживаемость после зимы | % | ключ к весеннему решению |
| RAI-008 | обеспеченность серой | мг/кг и кг/га | сильный фактор отклика рапса |
| RAI-009 | обеспеченность бором | мг/кг и кг/га | сильный фактор отклика рапса |

## 4. Минимальный стартовый пакет для обучения ядра

| Приоритет | Набор | Минимум |
|---|---|---|
| `P0` | статистика урожайности | 5–10 лет по регионам и районам |
| `P0` | погодные ряды | температура, осадки, экстремумы |
| `P0` | сортоиспытания | результаты по зонам |
| `P0` | агрохимия и почвы | `pH`, питание, типы почв |
| `P0` | нормативы и response-данные | сроки сева, дозы, защита |
| `P1` | обезличенные полевые истории | поле × сезон × технология × урожай |
| `P1` | наблюдения по болезням и вредителям | вид, степень, фаза, исход |
| `P1` | перезимовка и весенний старт | % выживаемости, дата возобновления |

## 5. Что нельзя включать в обучающий корпус без контроля

| Код | Недопустимый источник | Почему |
|---|---|---|
| X-001 | данные без происхождения | нельзя проверить качество и право использования |
| X-002 | статьи без исходных чисел | нет обучающих признаков |
| X-003 | таблицы без единиц | численные признаки невалидны |
| X-004 | агрегаты без методики расчёта | невозможно сопоставление |
| X-005 | пользовательские наблюдения без контекста | высокий риск шума |
| X-006 | смешанные региональные и полевые значения без разделения | ломается семантика модели |

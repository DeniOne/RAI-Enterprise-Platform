# Gamma: Recommendation UI

## Цель
Единый патент отображения рекомендаций для Telegram и Web-интерфейсов.

## Обязательные элементы
- Карточка рекомендации.
- Кнопки «Принять» и «Отклонить».
- Причина отклонения (фидбек).
- Ссылка на объяснение (Explainability).
- Ссылка на аудит-трейл.

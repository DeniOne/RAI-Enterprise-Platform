# Меню 5: GR

## 1. Бизнес-роль

Правовой и регуляторный контур: требования, ограничения, комплаенс и юридически значимые решения.

## 2. Целевое дерево страниц

- `/gr`
- `/gr/regulatory`
- `/gr/limits`
- `/gr/contracts`
- `/gr/compliance`
- `/gr/decisions`

## 3. Фактический статус

- Есть только `/gr` (заглушка).
- Подроуты отсутствуют.

## 4. API-связки

- `/api/gr*`
- `/api/legal*`
- `/api/consulting/decisions/*`

## 5. Критерий готовности MVP

- Юридический контур показывает: список требований, статус соблюдения, риски и реестр подтверждённых решений.

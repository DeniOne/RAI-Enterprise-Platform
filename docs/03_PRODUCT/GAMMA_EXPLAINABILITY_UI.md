# Gamma: Explainability UI

## Цель
Обеспечить понимание и доверие пользователя к выданным ИИ советам через прозрачность логики.

## Блоки
- Раздел «Почему система так решила» (факторы влияния).
- Визуализация уверенности модели (Confidence score).
- Ссылки на аналоги в прошлом («Как это было раньше»).

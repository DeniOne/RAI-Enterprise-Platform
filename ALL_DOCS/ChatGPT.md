# TRIAGE — Карта неизвестного (Input Map)

## 1️⃣ Обязательные входные данные (без них техкарта не формируется)

| Уровень     | Параметр                    | Ед. изм. | Диапазон      | Источник      | Кто подтверждает | Импутация допустима   |
| ----------- | --------------------------- | -------- | ------------- | ------------- | ---------------- | --------------------- |
| LegalEntity | ИНН/рег. данные             | string   | —             | договор       | директор         | ❌                     |
| Farm        | Регион / климатическая зона | enum     | predefined    | гео + профиль | агроном          | ⚠️ (по геокоординате) |
| Field       | Площадь                     | га       | >0            | кадастр/GPS   | агроном          | ❌                     |
| Field       | Координаты                  | lat/long | —             | GPS           | агроном          | ❌                     |
| Soil        | Тип почвы                   | enum     | классификатор | анализ        | лаб.             | ⚠️ (регион. профиль)  |
| Soil        | pH                          | ед.      | 3.5–8.5       | анализ        | лаб.             | ❌                     |
| Soil        | N, P₂O₅, K₂O                | мг/кг    | >0            | анализ        | лаб.             | ❌                     |
| Predecessor | Культура                    | enum     | —             | севооборот    | агроном          | ❌                     |
| Seed        | Гибрид                      | string   | —             | поставщик     | агроном          | ❌                     |
| Seed        | Норма высева                | кг/га    | >0            | расчёт        | ИИ+агроном       | ❌                     |
| Machinery   | Наличие техники             | список   | —             | хозяйство     | механик          | ❌                     |
| Labor       | Ответственные лица          | список   | —             | хозяйство     | директор         | ❌                     |

---

## 2️⃣ Желательные данные (повышают точность модели)

| Параметр                      | Ед. изм.   | Использование | Импутация |
| ----------------------------- | ---------- | ------------- | --------- |
| История урожайности (3–5 лет) | т/га       | baseline      | ⚠️        |
| Глубина гумуса                | см         | питание       | ⚠️        |
| Сера (S)                      | мг/кг      | питание рапса | ⚠️        |
| Бор (B)                       | мг/кг      | фено-критично | ⚠️        |
| Влага в почве                 | %          | срок сева     | ⚠️        |
| История болезней              | текст/enum | риск          | ❌         |

---

## 3️⃣ Импутируемые данные (с пометкой confidence)

* Климатические коэффициенты (через региональный профиль)
* Вероятность перезимовки (%)
* Давление вредителей (по региональной модели)
* Окна операций (через накопление GDD)

Каждое импутируемое поле получает:

```
provenance: "regional_model_vX"
confidence: 0.6–0.85
```

```
LegalEntity {
  id: UUID,

```

```
  id: UUID,
  area_ha: number (0–10000),

  soilType: enum,
  soilAnalysis: SoilReport,
  predecessor: enum
}
```

```
TechMap {
  id: UUID,
  season: string,
  crop: enum,
  targetYield_t_ha: number (0–10),
  hypothesisAssumptions: text,
  basePlan: Operation[],
  adaptiveRules: Trigger[]

```

```
Operation {
  id: UUID,
  type: enum,
  phase: enum,
  plannedWindow: {
    start: GDD or calendarRange,
    end: GDD or calendarRange

  norm: number,
  unit: string,
  dependencies: OperationID[],
  critical: boolean
}
```

---

## Двухконтурность

**Контур 1 — Базовый план**
Фиксированные операции + окна.

**Контур 2 — Триггеры**

| Триггер            | Порог     | Действие          |
| ------------------ | --------- | ----------------- |
| NDVI ниже baseline | -15%      | доп. обследование |
| Осадки > X мм      | логика    | перенос обработки |
| Фаза > допустимой  | нарушение | блок операции     |

---

## Ошибки

* Смесь несовместимых СЗР
* Превышение нормы >10%
* Пропуск критической фазы
* Нарушение окна GDD
